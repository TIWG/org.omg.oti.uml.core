trait UMLFactory[Uml <: UML] extends AnyRef

UML Factory

For the OTI xmi:ID & xmi:UUID ID generation algorithm to work properly, it is important that new elements be properly "owned" in the scope of a SerializableDocument. That is, a new element is either: - a root of a SerializableDocument (ie., a Package, a Profile or a Model) - an element that, right after being created, is added as an owned element of some other element that is in scope of a SerializableDocument (that is, by calling one of the owner's links_<assoc>_<metaclass1>_<end1>_compose_<end2>_<metaclass2> methods
